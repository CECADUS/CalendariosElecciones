# CalendariosElecciones

Web estática para GitHub Pages que calcula calendarios electorales de delegación de grupo y de curso, muestra las fechas en pantalla y rellena las plantillas PDF ubicadas en `docs/templates/`.

## Despliegue

Publica la carpeta `docs/` en GitHub Pages.

## Notas de cálculo

- REOREUS: artículos 31 y 32.
- RGREUS: título I y artículo 17.
- En delegación de grupo, los plazos abiertos del RGREUS se cierran usando el último día hábil permitido.
- En delegación de curso, la fecha de convocatoria se introduce manualmente.
